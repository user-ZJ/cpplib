# 刷题记录

### 1. 反转链表

```cpp
ListNode* ReverseList(ListNode* pHead) {
    ListNode *pre = nullptr;
    ListNode *cur = pHead;
    ListNode *nex = nullptr; // 这里可以指向nullptr，循环里面要重新指向
    while (cur) {
        // 反转next指针
        nex = cur->next;
        cur->next = pre;
        // 更新指针位置
        pre = cur;
        cur = nex;
    }
    return pre;
}
```
